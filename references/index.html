關於 5Q Prompt

5Q Prompt 是一個基於東方傳統哲學「五德」（木、火、土、金、水）核心架構研發的 AI Prompt 深度診斷工具。它不單純依賴關鍵字匹配，更引入了先進的語意推論（Semantic Inference）技術，全方位評估 Prompt 的結構、創意、清晰度、精準度與流暢度。

本工具由 theContext RD 團隊開發，旨在幫助開發者與生成式 AI 使用者，精準標定 Prompt 訊號，優化生成結果。

「五德」診斷架構

我們將 Prompt 的不同維度映射至五行屬性，透過獨家的評分演算法與「相生邏輯（Sheng Cycle）」校準，計算出最優化的「混合分數」。

🪵 木 (Wood)

結構與成長
評估 Prompt 的層次、框架與擴展性。

🔥 火 (Fire)

創意與張力
評估 Prompt 的創新性、情感與表達強度。

🧱 土 (Earth)

清晰與落地
評估 Prompt 的具體程度、背景與穩定性。

⚔️ 金 (Metal)

精準與剪裁
評估 Prompt 的約束、邏輯與簡潔度。

💧 水 (Water)

流暢與應變
評估 Prompt 的語句通順度與語境適應力。

🛡️ 100% 本地推論： 所有的語意分析與向量計算皆在您的設備上完成。您的 Prompt 內容絕不上傳至任何雲端伺服器，確保最高等級的開發隱私與資料安全。

核心技術與功能

  • 統一主入口 (Main Entry)： 整合多種推論引擎，靈活切換。
  • 高效能語意引擎： 支援 Candle (Pure Rust)、ONNX Runtime (ORT) 與 Burn (WGPU)。
  • 混合診斷模式 (Hybrid)： 完美結合傳統 Regex 關鍵字匹配與現代向量語意分析。
  • 三方語言解耦： 獨立控制 UI 語言 (-l)、語意錨點 (-a) 與 Regex 字典 (-r)。
  • 5QD 守護進程 (Daemon)： 支援背景服務模式，大幅加速語意推論速度。
  • 完善的授權與語言包： 內建 Tier 授權系統，支援擴充語言風格包 (.5qpack)。

快速開始 (CLI 範例)

5Q Prompt 是一個強大的命令行工具 (CLI)。以下是一些基本的診斷命令：

1. 基礎hybrid診斷 (中文 UI + 中文字典)

fiveq --prompt "請幫我寫一篇關於量子力學的科普文章"

2. 純語意分析模式 (Candle/ORT/Burn Backend)

fiveq -m semantic --prompt "Write a short story about a robot learning to love."

3. 國際化解耦呼叫 (繁體 UI + 中文錨點 + 中文字典)

fiveq -l hant -a zh -r zh --prompt "..."

💡 更多進階功能（如 5q daemon start 或 5q pack install），請參閱內建說明或開發文件。
